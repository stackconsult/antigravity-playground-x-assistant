# Changelog

All notable changes to the `antigravity-playground` (Ground Zero) will be documented in this file.

## [2.3.0] - 2026-02-09

### Added

- **Repository Hygiene**: Added `.gitignore`, `.env.example`, `LICENSE`, and `CONTRIBUTING.md`.
- **Tooling**: Integrated `ESLint` and `Prettier` for deterministic code quality.
- **Modularity**: Refactored `src/index.js` into a class-based plugin system.
- **CI/CD**: Enhanced `.github/workflows/agentic-verify.yml` for production validation.

### Changed

- **Kernel**: Updated `AGENTS.md` and `TRAINING_MANUAL.md` to production master standards.
- **Docs**: Polished `README.md` and `ARCHITECTURE.md` for clearer agentic onboarding.

## [2.2.0] - 2026-02-08

### Added

- Initial framework scaffold.
- Basic integrity test suite.
